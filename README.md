# super joao

an original game made in scratch for harvard's cs50 course.

collect cherries, avoid the bird. you have 3 lives.

pick enough cherries and the bird freezes in place as a ghostly copy of itself.

## how to play

1. download `superjoao.sb3`
2. go to https://scratch.mit.edu/projects/editor/
3. file > load from your computer
4. click the green flag

arrow keys to move.
